### 📌 SAI
*2025.08 ~ Present*

💻 [Source Code](https://github.com/Yonsei-Demo3/BE_API.git)  
📄 [API 명세서]()  
📑 [서비스 소개자료](/files/)

**Description.**  
사용자가 생성한 Q&A/토론방의 대화 내용을 자동 요약하고 핵심 논점을 정리해주는 AI 기반 소통 플랫폼입니다.  
주요 목적은 복잡한 대화 흐름을 정리하고, 토론의 주요 논지를 빠르게 파악할 수 있도록 돕는 것입니다.

**Features.**  
- Spring Boot 기반 API 서버 구축  
- 회원가입/로그인(JWT) 및 세션 기반 접근 제어  
- Q&A/토론방 CRUD 기능 구현  
- Redis 기반 세션 캐싱 및 인증 토큰 관리  
- OpenAI API 기반 대화 요약 기능 프로토타입 완성  
- 관리자용 모니터링 엔드포인트 설계

**Tech Stack.**  
`Spring Boot` `MySQL` `Redis`  
`JWT` `OpenAI API`  
`Docker` `AWS EC2`
