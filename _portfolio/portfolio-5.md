### 📌 SAI
*2025.08 ~ Present*

💻 [Source Code](https://github.com/Yonsei-Demo3/BE_API.git)  
📑 [서비스 소개자료](/files/sai.pdf)

**Description.**  
사용자가 생성한 Q&A·토론방의 대화를 구조적으로 관리하고, 핵심 메시지를 빠르게 파악할 수 있도록 돕는 토론 지원 플랫폼입니다.
각 토론방의 질문, 답변, 대화 흐름을 명확히 정리하여 사용자가 토론의 맥락을 손쉽게 이해하도록 하는 데 목적이 있습니다.

**My Contributions**  
- Spring Boot 기반 API 서버 구축  
- 회원가입/로그인(JWT) 및 세션 기반 접근 제어  
- Q&A/토론방 CRUD, 대화 로그 저장 구조 설계 및 API 개발
- Redis 기반 세션 캐싱 및 인증 토큰 관리 최적화
- Docker, EC2 기반 배포 환경 설정
- GitHub Actions를 이용한 CI/CD 자동화 파이프라인 구축

**Tech Stack.**  
`Spring Boot` `MySQL` `Redis`  
`JWT` `OpenAI API`  
`Docker` `AWS EC2`
`Github Actions (CI/CD)`
